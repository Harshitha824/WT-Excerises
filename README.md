# 🌐 WT Exercises Repository

## 📌 About

This repository contains all the **Web Technology (WT) Lab Exercises** completed as part of the **B.Tech Computer Science and Engineering (2023)** curriculum at JAIN University.

It includes basic to intermediate web development concepts using **HTML, CSS, JavaScript, PHP, and MySQL**.

---

## 📚 Exercises Included

### 🔹 Basic Web Development

1. Create a personal website with a minimum of 4 web pages using basic HTML tags.
2. Create a static E-commerce website using advanced HTML tags.

### 🎨 Styling with CSS

3. Enhance Exercise-1 using Inline, Internal, and External CSS.
4. Enhance Exercise-2 using Inline, Internal, and External CSS.

### ⚙️ JavaScript

5. Add a scientific calculator as a menu in Exercise-1 using JavaScript.
6. Create registration and login forms for Exercise-2 with validation using JavaScript.

### 🔄 Advanced Concepts

7. Implement event handling mechanisms (Exercises 1 & 2).
8. Perform form handling and validation using PHP.
9. Create, store, retrieve, and display product details using PHP & MySQL.
10. Implement sessions and cookies across multiple pages.

## 📁 Project Structure

* `portfolio.html` – Personal website
* `ecommerce.html` – E-commerce site without style
* `exercise*.html` – Lab exercises
* `index.html` – Main entry page for E-commerce
* `style.css` – Styling file for E-commerce

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/WT-Exercises.git
   ```
2. Open the folder in a browser or VS Code.
3. Run HTML files directly in your browser.
4. For PHP/MySQL exercises, use XAMPP or a local server.
